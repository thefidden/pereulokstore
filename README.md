# Магазин "Переулок"
Данный репозиторий содержит основные файлы, участвующие в функционировании серверной и клиентских логик магазина "Переулок".

## SERVER/APP
Каталог содержит в себе файлы серверной логики, построенной на фреймворке Django языка программирования Python.

## CLIENT-VUE/APP
Каталог содержит в себе файлы клиентской логики, построенной на фреймворке Vue.js языка программирования Typescript. Был использован сборщик Vite и пакетный менеджер Yarn.

## CLIENT-REACT-TYPESCRIPT/APP
Каталог содержит в себе файлы клиентской логики, построенной на фреймворке React языка программирования Typescript. Был использован сборщик Vite и пакетный менеджер Yarn.
